# Learning: ng-iac repo skill discovery path

## Context
In `/Users/jnyjc2/code/ng-iac`, the user asked why the local `iac-mr-review` skill did not appear when typing `$` in the Codex app. The skill had been stored under a top-level `skills/` directory.

## Learning
Codex scans repo-local skills from `.agents/skills`, not from a top-level `skills/` folder. The `iac-mr-review` skill was moved intact to `/Users/jnyjc2/code/ng-iac/.agents/skills/iac-mr-review`, including its `SKILL.md` and `agents/openai.yaml` metadata.

## Future Use
For future work in this repo, expect `$iac-mr-review` to be discoverable from `.agents/skills/iac-mr-review` after Codex reloads skills. If it does not appear in `$` autocomplete, advise restarting Codex or opening a new thread before changing the skill contents.
