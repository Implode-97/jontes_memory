# Learning: Spark 4.1 and 4.2 ML/ETL feature baseline

## Context
While researching current Apache Spark 4.1 and upcoming 4.2 information for the RAPIDS innovation-week workspace, official Apache news, Spark docs, and Spark JIRA were checked on 2026-06-16.

## Learning
As of 2026-06-16, Spark 4.1.2 is the stable 4.1 line and Spark 4.2.0 is still preview-only, with preview5 announced on 2026-05-01. For ML/ETL work, Spark 4.1's most relevant stable changes are Spark Declarative Pipelines, Structured Streaming Real-Time Mode for Scala stateless workloads, Arrow-native Python UDF/UDTF work, Python data-source filter pushdown, Spark ML on Connect GA for Python, and VARIANT shredding. Spark 4.2 items to track include metric views, CDC/AutoCDC-related JIRA work, PySpark RTM support, Arrow-optimized Python UDFs enabled by default, DSv2 ordering/shuffle avoidance, vector/nearest-neighbor SQL work, and language-agnostic UDF protocol work.

## Future Use
When advising RAPIDS or Databricks/Spark users, separate Spark 4.1 stable recommendations from Spark 4.2 preview/watch-list items. Re-check Apache release notes before treating any 4.2 capability as production-ready.
