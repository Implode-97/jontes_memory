# Learning: CAV-134396 Data Foundation onboarding epic

## Context
The user asked Codex to read Jira epic `CAV-134396` via the Jira MCP while working in the sandbox catalog workspace.

## Learning
`CAV-134396` is the PI2616 successor-style epic for "Data Foundation - Ingestion-team-onboarding". It builds on `CAV-130873` and shifts the Databricks dev workspace work from initial platform readiness toward governed, repeatable team onboarding. The epic scope includes external locations with storage credentials, instance profiles, SQL warehouse usage, guardrails, user-facing roles, onboarding Woodpeckers, cost visibility tagging, workspace settings, and TOOCAN validation. The acceptance criteria emphasize Frankfurt dev workspace readiness, documented/validated platform capabilities, defined onboarding model, secure guardrails, one end-to-end team onboarding flow, and permission validation.

## Future Use
When planning Data Platform work under this epic, treat onboarding, validation, and governance as the core goal. Search Jira for child work under `CAV-134396`; as of the initial read, it had no comments and no child issues found by Epic Link/parent search.
