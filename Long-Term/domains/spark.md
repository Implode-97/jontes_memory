# Spark Release Watch

- June 2026 baseline: Spark 4.1.2 was the stable 4.1 line and Spark 4.2.0 remained preview. Re-check Apache release notes before production advice.
- Spark 4.1 topics included declarative pipelines, Scala stateless streaming real-time mode, Arrow-native Python UDF/UDTF work, Python data-source pushdown, Spark ML on Connect GA for Python, and VARIANT shredding.
- Spark 4.2 watch items included metric views, CDC/AutoCDC, PySpark real-time mode, default Arrow-optimized Python UDFs, DSv2 ordering/shuffle work, vector SQL, and language-agnostic UDF protocols.
