# Spark Memory

## Spark Releases

- As of 2026-06-16, treat Spark 4.1.2 as the stable 4.1 line and Spark 4.2.0 as preview-only, with preview5 announced on 2026-05-01. Re-check Apache release notes before treating a Spark 4.2 capability as production-ready.
- For ML/ETL advice, separate stable Spark 4.1 items from 4.2 watch-list items. Spark 4.1 stable highlights include Spark Declarative Pipelines, Structured Streaming Real-Time Mode for Scala stateless workloads, Arrow-native Python UDF/UDTF work, Python data-source filter pushdown, Spark ML on Connect GA for Python, and VARIANT shredding.
- Spark 4.2 items to track include metric views, CDC/AutoCDC-related work, PySpark Real-Time Mode support, Arrow-optimized Python UDFs enabled by default, DSv2 ordering/shuffle avoidance, vector or nearest-neighbor SQL work, and language-agnostic UDF protocol work.
